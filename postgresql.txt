패키지 관리 명령어 도구 업뎃
sudo apt-get update 

postgresql 관련 버전 확인
sudo apt search postgresql

postgresql 패키지 실행 
sudo apt install postgresql

postgresql 경로 
cd /etc/postgresql

깔린 버전 확인 
ls

외부접속 파일 경로 
cd /10/main

수정 파일 
vi postgresql.conf // 로컬호스트 부분 * -> 모두 접속으로 변경 
vi pg_hba.conf // 0.0.0.0/0 -> 모든 아이피 허용 

서비스 재시작
sudo service postgresql restart
sudo service postgresql stop
